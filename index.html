Why Baseline

Built to feel straightforward for users, while staying verifiable for everyone.

Open participation

Anyone can participate. No accounts, no “invites”, no approval process—just an open network.

Transparent by default

Activity is publicly verifiable. The goal is clarity: predictable behavior, fewer surprises.

Practical payments

Standard sends for everyday transfers. Optional Scheduled Sends when “send later” matters.

For miners & builders (advanced)

Mining rigs

Baseline can be secured by miners, and rigs can connect to a node’s mining endpoint. If you’re a miner, this is where you plug in your setup.

Operators

Nodes expose an API for integrations and tooling, with authentication and sensible limits to keep the service stable.

Technical terms

The network uses Proof-of-Work. Mining integrations typically use Stratum. Tooling uses JSON-RPC.

Payments

Keep it simple for everyday usage, with optional time-based intent when you need it.

  • Standard sends for normal payments.
  • Scheduled Sends (optional): pick a future UTC date/time.
  • Cancelable or non-cancelable scheduling modes depending on the use-case.
  • Verify status in wallet and explorer.

Earn by securing the network

If you run mining hardware, you can connect it to a Baseline node and contribute security.

Run a node

Keep a node online so it can stay synced and serve mining work to connected hardware.

Connect hardware

Point your rig at the node’s mining endpoint. Use your payout address so rewards go where you want them.

Track payouts

Rewards mature over time. The operator flow is designed to make payouts predictable and auditable.

Tools

Baseline includes the basics to participate and verify: a wallet, an explorer, and an API for integrations.

Wallet

Send and receive, with optional Scheduled Sends.

Explorer

Browse blocks and transactions, plus schedules when relevant.

API

JSON-RPC for tools, integrations, and operators (advanced).
