Quick Start

Start a node, open the wallet, and (optionally) connect a miner.

Run a node Local 
git clone https://github.com/Baseline-Blockchain/baseline-node.git
cd baseline-node
py -3 -m venv .venv
.venv\Scripts\activate
pip install -e .
baseline-node --config config.json
git clone https://github.com/Baseline-Blockchain/baseline-node.git
cd baseline-node
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
baseline-node --config config.json

Update RPC credentials in config.json before exposing anything to the internet.

Open the wallet GUI 
baseline-wallet-gui

The wallet talks to your local node via JSON-RPC (start the node first).

Requires Python's Tkinter GUI libs (bundled on Windows/macOS; on Linux install python3-tk or python3-tkinter).

Light wallets are coming soon. Meanwhile, the GUI works as a full node wallet.

Mine (optional) Stratum 
cpuminer-multi -a sha256d -o stratum+tcp://127.0.0.1:3333 -u YOURADDRESS.worker1 -p x

Requires Stratum enabled in your node config (set mining.pool_private_key). See README Mining.
