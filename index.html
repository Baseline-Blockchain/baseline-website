Quick Start

Choose your path: light wallet (no node required) or full node + optional mining.

Use the light wallet Desktop
  • Download: Latest release (Win/macOS/Linux).
  • Keys stay local: Create or import; signing happens on-device.
  • Does not support scheduled sends: Use a full node for that feature and it's built-in wallet.

Roadmap

Baseline Cash is built for rule stability, verifiability, and a miner-first security story. This roadmap is intentionally conservative: we ship improvements only when they strengthen security and keep the protocol minimal.

Now Next releases
  • Consensus & policy spec: versioned rules document + clear divergences.
  • Test vectors: hashing/byte-order, LWMA difficulty, coinbase/dev-fund, schedules.
  • Network status: hashrate estimate, orphan rate, reorg log, difficulty view.
  • Mining UX: “mine in 10 minutes” + “run your own pool” defaults and docs.
Next Operator-first
  • Node hardening: safe-by-default config templates + RPC exposure guidance.
  • Monitoring: metrics/log formats + alerting recommendations.
  • Release discipline: deterministic build notes, upgrade guides.
  • Explorer primitives: better reorg visualization and fee/mempool stats.
Later No bloat, no fork treadmill
  • Propagation improvements: reduce orphan risk without bloating consensus.
  • Difficulty edge-case hardening: clamp/guardrails based on real-world data.
  • Miner/pool auditability: transparency dashboards and reference tooling.
  • Ecosystem glue: exchange integration docs + opt-in pool/node directory.
Non-goals (what we won’t do)
  • No platform pivot: Baseline Cash is not trying to become a general smart-contract/DeFi chain.
  • No fork treadmill: we won’t change PoW on a schedule; changes happen only for clear security reasons.
  • No governance bloat: decisions stay transparent and off-chain; consensus changes require broad ecosystem agreement.
