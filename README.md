# Baseline Marketing Site

Static single-page site in `marketing/` (no build step).

## Preview locally

- Open `index.html` directly in a browser, or
- Run a tiny local server from the repo root:
  - `python -m http.server 8080`
  - Visit `http://127.0.0.1:8080`

## Assets

- Logo: `assets/logo.png` (copied from the explorer).
- Icons: Bootstrap Icons (vendored) in `assets/bootstrap-icons/` (see `assets/bootstrap-icons/LICENSE`).
